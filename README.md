# ER Time - Real-Time Emergency Room Wait Times

![ER Time Logo](https://img.shields.io/badge/ER%20Time-Real--Time%20Wait%20Times-blue?style=for-the-badge&logo=medical-cross)

A revolutionary mobile and web application that provides real-time emergency room wait times, helping patients find the fastest emergency care when it matters most.

## 🚀 Features

- **Real-Time Wait Times**: Get accurate, up-to-date emergency room wait times
- **GPS Hospital Finder**: Find nearest emergency rooms with turn-by-turn directions
- **AI-Powered Predictions**: Advanced algorithms predict accurate wait times
- **Offline Access**: Works without internet connection
- **Community Reviews**: Read and share real patient experiences
- **Privacy First**: Your health data is encrypted and secure

## 🛠️ Technology Stack

- **Frontend**: Flutter (Cross-platform)
- **Backend**: Django REST API
- **Database**: PostgreSQL with AI-powered deduplication
- **Real-time**: WebSocket connections for live updates
- **Maps**: Google Maps API + TomTom integration
- **AI/ML**: Custom algorithms for wait time prediction

## 🔒 Privacy & Security

- **Data Encryption**: All data encrypted in transit and at rest
- **Privacy First**: No personal health records collected

## 🚀 Getting Started

### For Users
1. Download the app from your preferred platform
2. Allow location access for nearby hospital search
3. Browse real-time wait times
4. Get directions to your chosen hospital
5. Share your experience to help others

### For Developers
```bash
# Clone the repository
git clone https://github.com/zub165/ERtimeold.git
cd ERtimeold

# Install Flutter dependencies
flutter pub get

# Run the app
flutter run
```
